# Java Email Sender
Projeto Simples de como enviar e-mail utilizando o serviço de Email da Amazon (Amazon Simple Email Service - SES) com mensageria e arquitetura de microsserviço. 

## tecnologias:
* Springboot
* SpringData
* SpringCloud
* Postgres
* RabbitMQ
* Maven
